Connecting Interactive Brokers — Python Example

from ib_async import IB, Stock, Future, util

# 1. Connect (TWS must be running on port 7497)
ib = IB()
ib.connect('127.0.0.1', 7497, clientId=1)

# 2. Fetch AAPL 1H bars (last 30 days)
contract = Stock('AAPL', 'SMART', 'USD')
bars = ib.reqHistoricalData(
    contract, endDateTime='', durationStr='30 D',
    barSizeSetting='1 hour', whatToShow='MIDPOINT', useRTH=True
)
df_1h = util.df(bars)   # → resample to 4H in live-server.py

# 3. ES Futures
es_contract = Future('ES', '202503', 'CME')
bars_es = ib.reqHistoricalData(es_contract, ...)

# 4. Subscribe to real-time ticks
ib.reqMktData(contract, '', False, False)

4-Hour Bar Construction

import yfinance as yf

# Download 1H bars (90-day window)
raw = yf.download('AAPL', period='90d', interval='1h')

# Resample to 4H candles
df_4h = raw.resample('4h').agg({
    'Open':   'first',
    'High':   'max',
    'Low':    'min',
    'Close':  'last',
    'Volume': 'sum'
}).dropna()
# Each bar = 4 hours of price action — ~6 bars per trading day

🚀 Running the Live Trading Server

  1. Install Python dependencies:
    pip install flask flask-cors yfinance scikit-learn pandas numpy
  2. Start the server (from the Machine Learning project folder):
    python live-server.py
  3. Serve the website (from the website folder):
    npx serve -l 8080 website/
  4. Open http://localhost:8080/live-trading.html — the dashboard auto-connects to localhost:5050.
  5. Upgrade to live data: Replace yfinance with ib_async and connect Interactive Brokers TWS on port 7497.

Server Endpoints

/api/candles/AAPL 4H OHLCV array for any symbol
/api/signal/ES=F ML direction prob + magnitude + trade params
/api/multi/AAPL,ES=F,GC=F Batch signals for multiple symbols
/api/symbols List of all available symbols
