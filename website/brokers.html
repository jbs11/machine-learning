How to Get Your Free API Keys

α Alpaca (Recommended)
  1. Go to app.alpaca.markets/signup
  2. Sign up with email (no funding required)
  3. Switch to Paper Trading account
  4. Click View → API Keys
  5. Generate & copy Key ID + Secret Key
T Tradier
  1. Go to developer.tradier.com
  2. Sign up for a developer account
  3. Go to Applications → New Application
  4. Generate your Access Token
  5. Use sandbox checkbox for free access
F Finnhub
  1. Go to finnhub.io/register
  2. Sign up with email only
  3. No credit card required
  4. API key shown immediately on dashboard
  5. 60 calls/min free forever
